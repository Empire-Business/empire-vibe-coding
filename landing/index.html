O que é

Estrutura para quem constrói com IA

Uma skill que você ativa no Claude Code e que transforma qualquer projeto numa máquina de desenvolvimento organizada, segura e documentada.

⚙️

Setup Automático

Instala o CLAUDE.md padrão Empire no seu projeto. Se você já tem um, faz merge inteligente — sem apagar nada do que você construiu.

🔐

Segurança Nativa

Garante que a skill /security-auditor está instalada e atualizada globalmente. Auditoria obrigatória antes de qualquer deploy.

📋

Tasks 100% do Tempo

Toda ação cria tasks visíveis antes de executar. Você sempre sabe exatamente o que o Claude está fazendo — e por quê.

📚

Documentação Viva

Toda doc vai para docs/. O CLAUDE.md é um índice enxuto. Ao remover código, a doc morre junto — sem lixo acumulando.

🔄

Auto-atualização

Peça "verifique atualizações" e a skill faz git pull, atualiza o security-auditor se necessário e reporta o que mudou.

🏗️

Stack Definida

React 18 + TypeScript + Vite + Tailwind + shadcn/ui + Supabase + Vercel. Nenhuma decisão de arquitetura fica sem documentar.

Como funciona

Da primeira ativação ao deploy

A skill detecta automaticamente em qual fase seu projeto está e age de forma correspondente.

01

Ativar

Digite /empire-vibe-coding no Claude Code. A skill lê o estado do projeto.

02

Setup

Instala CLAUDE.md, verifica o security-auditor e cria o state document .empire/state.json.

03

Codar

Em cada pedido, cria tasks, lê o CLAUDE.md e executa seguindo as regras definidas.

04

Deploy

Audita segurança obrigatoriamente, documenta o deploy e atualiza o CHANGELOG.

Instalação

Três passos para começar

Funciona em qualquer projeto. Não importa se você está começando do zero ou já tem código rodando.
Works with any project — new or existing.

1

Clone o repositório na pasta de skills

Baixe a skill direto para onde o Claude Code procura por skills globais.

# Clone into Claude Code global skills directory git clone https://github.com/Empire-Business/empire-vibe-coding \ ~/.claude/skills/empire-vibe-coding
2

Abra seu projeto no Claude Code

Navegue até a pasta do seu projeto (novo ou existente).

# Open your project cd meu-projeto claude
3

Ative a skill

A skill faz todo o resto: detecta o estado, instala CLAUDE.md, verifica o security-auditor.

/empire-vibe-coding

Requisitos
Claude Code instalado · Git · macOS ou Linux

Para manter atualizado
Peça ao Claude: "verifique atualizações da skill" — ele faz git pull e atualiza o security-auditor se necessário.

Projeto existente com CLAUDE.md?
Sem problema. A skill faz merge — preserva tudo que você já tem e adiciona apenas o que está faltando.
